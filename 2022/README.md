# Building & running instructions

Commands to run to build and run each implementation.

- Day 1

  ```bash
  javac day1.java && java day1
  ```

- Day 2

  ```bash
  node day2.js
  ```

- Day 3

  ```bash
  python3 day3.py
  ```

- Day 4

  ```bash
  gcc day4.c -o day4 -std=c17 && ./day4
  ```

- Day 5

  ```bash
  gcc day5.cpp -o day5 -std=c++20 -lstdc++ && ./day5
  ```

- Day 6

  ```bash
  tsc && node day6.js
  ```
